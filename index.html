原版地址

# 内容简介

在浏览器中提供丰富的音频和视频实时通信和点对点数据交换，不需要私有插件。这个简明的实践指南向您展示了如何使用新兴的Web实时通信(WebRTC)技术来构建一个浏览器到浏览器的应用程序。

作者通过实例学习的方法非常适合那些希望理解实时通信的 web 程序员，以及不熟悉 HTML5 和基于 javascript 的客户端-服务器 web 编程的电信架构师。
